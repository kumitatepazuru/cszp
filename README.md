## Download
### Note This program is premised on running on linux (Debian system). So it may not run on other OS.
Operation check OS
- ubuntu 16.04LTS
- ubuntu server 16.04LTS
- debian 10.1

## How to install

#### 1.Get this program.
```git clone https://github.com/kumitatepazuru/cszp.git```
#### 2.Install
``` 
cd ./cszp/
pip3 install cuitools
python3 install.py
```
